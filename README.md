## Running the Website

1. Clone the repo

```sh
git clone https://github.com/JackHumphries9/jackh.sh
```

2. Install NPM packages

```sh
npm install
```

To run this project in development mode, type

```sh
npm run dev
```

To build this project, run:

```sh
npm run build
```
